# Analisador de Vendas - Loja TechVision

Este projeto faz parte do meu portfólio de estudos em Análise e Desenvolvimento de Sistemas. O objetivo é simular o ecossistema de vendas de uma loja de eletrônicos para praticar a manipulação de dados com Python e Pandas.

## 🚀 Funcionalidades
- Estruturação de dados complexos usando dicionários Python.
- Processamento e limpeza de dados com a biblioteca **Pandas**.
- Cálculo de métricas de negócio:
  - Faturamento total por produto.
  - Desempenho de vendas por vendedor.
  - Monitoramento de níveis críticos de estoque.

## 🛠️ Tecnologias Utilizadas
- **Python 3.x**
- **Pandas** (Biblioteca para análise de dados)
- **PyCharm** (IDE)

## 📊 Estrutura do Projeto
- `main.py`: Arquivo principal com a lógica de análise.
- `estrutura_de_dados.py`: Módulo contendo o dicionário com as informações da loja.

## 📝 Como executar
1. Certifique-se de ter o Python e o Pandas instalados.
2. Execute o arquivo `main.py` para visualizar os relatórios no console.
